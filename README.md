# 03-IT-Support-Specialist-Lab

**Enterprise IT Support Specialist Lab | Windows Server | Active Directory | DNS | DHCP | RDP | Microsoft 365 | Ticketing | Troubleshooting**

This repository is a hands-on IT Support Specialist lab built to document the real duties, tools, and troubleshooting responsibilities found across Help Desk, Service Desk, Desktop Support, and IT Support roles.

This repo serves as one chapter of proof in my GitHub portfolio.

---

## Quick View

* Building job-ready IT Support skills through hands-on labs
* Based on common duties found across IT Support job postings
* Focused on Windows Server, Active Directory, DNS, DHCP, RDP, Microsoft 365, printers, file shares, endpoint support, and ticketing
* Designed to create both personal proof and a guided learning path for entry-level IT professionals
* Current status: Active / In Progress
